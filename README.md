# 📊 Lecture Feedback Analyzer

Інструмент для аналізу відгуків студентів на лекції освітніх програм.  
Завантажуєш Excel — отримуєш рейтинг лекцій, % розподіл оцінок і AI-інсайти з текстових відповідей.

**→ [Відкрити інструмент](https://maksymmokhun.github.io/lecture-feedback/lecture-analyzer.html)**

---

## Що робить

- **Рейтинг лекцій** — всі лекції відсортовані від найкращої до найгіршої за composite score (середнє трьох рейтингових питань)
- **% розподіл оцінок** — по кожному з трьох питань з варіантами відповідей
- **Порівняння з середнім по програмі** — для кожної лекції показує delta (+/−) відносно середнього
- **AI-інсайти** — аналіз текстових відповідей: що цінували студенти, що покращити лектору, нерозкриті питання, загальний тон

## Структура Excel-файлу

Кожна вкладка = одна лекція. Назва вкладки = назва лекції.

Колонки (в будь-якому порядку):

| Колонка | Тип | Варіанти |
|---|---|---|
| Оцінка лекції | Вибір | Сподобалась повністю / Сподобалась, але є зауваження / 50/50 / Не дуже сподобалась / Зовсім не сподобалась |
| Наскільки корисною була інформація | Вибір | Дуже корисна / Достатньо корисна / Складно сказати / Не дуже корисна / Зовсім не корисна |
| Чи комфортно тобі взаємодіяти з лектором | Вибір | Дуже комфортно / Достатньо комфортно / Складно сказати / Не дуже комфортно / Зовсім некомфортно |
| Що корисного дізнався/навички отримав | Текст | — |
| Що варто покращити лектору | Текст | — |
| Які питання не було розкрито | Текст | — |
| Коментар | Текст | — |

## Як користуватись

1. Відкрий **[посилання на інструмент](https://maksymmokhun.github.io/lecture-feedback/lecture-analyzer.html)**
2. (Опційно) Встав Anthropic API key для AI-інсайтів
3. Завантаж Excel-файл з відгуками
4. Клікай на лекції в лівому сайдбарі

## Технічний стек

- Чистий HTML + JS, без бекенду
- [SheetJS](https://sheetjs.com/) для парсингу Excel
- [Anthropic API](https://anthropic.com) для AI-аналізу текстових відповідей (claude-sonnet-4-6)
- Хоститься на GitHub Pages — безкоштовно, без сервера

## AI-інсайти

Для роботи AI-блоку потрібен Anthropic API key (`sk-ant-...`).  
Ключ вводиться перед завантаженням файлу і зберігається тільки в пам'яті браузера — нікуди не надсилається, крім api.anthropic.com.

Статистична частина (рейтинги, %, порівняння) працює без ключа.
