# 📊 Lecture Feedback Analyzer

Інструмент для аналізу відгуків студентів на лекції освітніх програм Genesis Academy.  
Завантажуєш Excel → отримуєш рейтинг лекцій, % розподіл оцінок, AI-інсайти. Звіти зберігаються в спільному архіві команди.

**→ [Відкрити інструмент](https://maksymmokhun.github.io/lecture-feedback/lecture-analyzer.html)**

> Доступно лише для акаунтів `@gen.tech` — браузер має бути залогінений під корпоративним Google акаунтом.

---

## Що робить

- **Рейтинг лекцій** — всі лекції відсортовані від найкращої до найгіршої за composite score (середнє трьох рейтингових питань)
- **% розподіл оцінок** — по кожному з трьох питань з варіантами відповідей
- **Порівняння з середнім по програмі** — delta (+/−) відносно середнього по всіх лекціях
- **AI-інсайти** — аналіз текстових відповідей: що цінували студенти, що покращити лектору, нерозкриті питання, загальний тон
- **Спільний архів команди** — збережені звіти доступні всім колегам з будь-якого девайсу

## Як користуватись

1. Відкрий **[посилання на інструмент](https://maksymmokhun.github.io/lecture-feedback/lecture-analyzer.html)** — переконайся що браузер залогінений під `@gen.tech`
2. (Опційно) Встав Anthropic API key для AI-інсайтів
3. Завантаж Excel-файл з відгуками
4. Клікай на лекції в лівому сайдбарі
5. Натисни **"💾 Зберегти звіт"** — він з'явиться в архіві у всіх колег

Щоб переглянути чужий звіт — клікни на нього в блоці "Архів звітів команди" на головному екрані.

## Структура Excel-файлу

Кожна вкладка = одна лекція. Назва вкладки = назва лекції.

| Колонка | Тип | Варіанти відповідей |
|---|---|---|
| Оцінка лекції | Вибір | Сподобалась повністю / Сподобалась, але є зауваження / 50/50 / Не дуже сподобалась / Зовсім не сподобалась |
| Наскільки корисною була інформація | Вибір | Дуже корисна / Достатньо корисна / Складно сказати / Не дуже корисна / Зовсім не корисна |
| Чи комфортно тобі взаємодіяти з лектором | Вибір | Дуже комфортно / Достатньо комфортно / Складно сказати / Не дуже комфортно / Зовсім некомфортно |
| Що корисного дізнався/навички отримав | Текст | — |
| Що варто покращити лектору | Текст | — |
| Які питання не було розкрито | Текст | — |
| Коментар | Текст | — |

Порядок колонок — будь-який. Назви — точні (інструмент шукає за ключовими словами).

## Технічний стек

| Компонент | Технологія |
|---|---|
| Фронтенд | Чистий HTML + JS, без фреймворків |
| Парсинг Excel | [SheetJS](https://sheetjs.com/) |
| AI-аналіз | [Anthropic API](https://anthropic.com) — `claude-sonnet-4-6` |
| Спільне сховище | Google Sheets + Apps Script (Web App) |
| Хостинг | GitHub Pages — безкоштовно, без сервера |

## AI-інсайти

Потрібен Anthropic API key (`sk-ant-...`). Вводиться перед завантаженням файлу, зберігається тільки в пам'яті браузера — нікуди не надсилається, крім `api.anthropic.com`.

Статистична частина (рейтинги, %, порівняння, архів) працює без ключа.

## Архів звітів

Звіти зберігаються в Google Sheet [`GA Lecture Analyzer DB`](https://docs.google.com/spreadsheets/d/1WEbiZoRI-RQSngSQLORbAwPthOuvEn8Euj_gb76P2j4).  
Вкладка `reports` — кожен рядок = один збережений звіт у форматі JSON.

Доступ до архіву вимагає авторизації через `@gen.tech` Google акаунт — запити йдуть через Apps Script Web App задеплоєний на тому ж домені.
