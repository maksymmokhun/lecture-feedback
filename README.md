# 📊 Lecture Feedback Analyzer

Інструмент для аналізу відгуків студентів на лекції освітніх програм Genesis Academy.  
Завантажуєш Excel → отримуєш рейтинг лекцій, % розподіл оцінок, AI-інсайти. Звіти зберігаються в спільному архіві команди.

**→ [Відкрити інструмент](https://maksymmokhun.github.io/lecture-feedback/lecture-analyzer.html)**

---

## Для команди: як почати

### Крок 1 — Відкрий інструмент
Перейди за посиланням вище.

### Крок 2 — Введи GitHub токен
При першому відкритті з'явиться екран налаштування. Введи токен який отримав від Макса і натисни **"Зберегти та продовжити"**.

Токен зберігається тільки у твоєму браузері — більше нічого робити не треба. Наступного разу одразу відкриється архів.

### Крок 3 — Завантаж файл з відгуками
Натисни **"Вибрати файл"** або перетягни Excel-файл у зону завантаження.  
Кожна вкладка в файлі = одна лекція.

### Крок 4 — Переглядай аналіз
Клікай на лекції в лівому сайдбарі. Для кожної лекції побачиш:
- % розподіл оцінок по трьох питаннях
- Порівняння з середнім по програмі
- AI-інсайти (якщо введено Anthropic API key)

### Крок 5 — Збережи звіт в архів
Натисни **"💾 Зберегти звіт"** у верхньому правому куті.  
Звіт з'явиться в архіві у всіх колег одразу.

### Перегляд архіву
На головному екрані в правому блоці — всі збережені звіти команди. Клікни на будь-який щоб відкрити. При перегляді архіву API не викликається — кошти не списуються.

## Структура Excel-файлу

Кожна вкладка = одна лекція. Назва вкладки = назва лекції.

| Колонка | Тип | Варіанти відповідей |
|---|---|---|
| Оцінка лекції | Вибір | Сподобалась повністю / Сподобалась, але є зауваження / 50/50 / Не дуже сподобалась / Зовсім не сподобалась |
| Наскільки корисною була інформація | Вибір | Дуже корисна / Достатньо корисна / Складно сказати / Не дуже корисна / Зовсім не корисна |
| Чи комфортно тобі взаємодіяти з лектором | Вибір | Дуже комфортно / Достатньо комфортно / Складно сказати / Не дуже комфортно / Зовсім некомфортно |
| Що корисного дізнався/навички отримав | Текст | — |
| Що варто покращити лектору | Текст | — |
| Які питання не було розкрито | Текст | — |
| Коментар | Текст | — |

Порядок колонок — будь-який. Інструмент визначає їх автоматично за ключовими словами.

## AI-інсайти (опційно)

Для генерації текстового аналізу потрібен Anthropic API key (`sk-ant-...`).  
Вводиться на головному екрані перед завантаженням файлу. Зберігається тільки в пам'яті браузера.

Без ключа статистична частина (рейтинги, %, порівняння, архів) працює повністю.  
При перегляді збережених звітів з архіву API не викликається — вже згенеровані інсайти зберігаються разом зі звітом.

## Технічний стек

| Компонент | Технологія |
|---|---|
| Фронтенд | HTML + JS, без фреймворків |
| Парсинг Excel | SheetJS |
| AI-аналіз | Anthropic API — claude-sonnet-4-6 |
| Спільне сховище | GitHub Gist (shared token) |
| Хостинг | GitHub Pages |

Інструмент працює повністю в браузері — без сервера, без бази даних, без деплою.

## Для адміна (Макс)

**Архів звітів** зберігається в GitHub Gist акаунту `maksymmokhun`, файл `ga-lecture-reports.json`.

**Якщо потрібно очистити архів** — відкрий [gist.github.com](https://gist.github.com), знайди файл `ga-lecture-reports.json`, натисни Edit, заміни вміст на `[]`, збережи.

**Якщо токен перестав працювати** (GitHub анулює токени автоматично після певних дій) — згенеруй новий на [github.com/settings/personal-access-tokens/new](https://github.com/settings/personal-access-tokens/new):
- Repository access: **No access**
- Permissions → Account → Gists: **Read and write**

Надішли новий токен команді в Slack — кожен натискає 🔑 на сайті і вводить новий.
